##### Инструкции по запуску:
- Установите и запустите [OpenServer](https://ospanel.io/)
- Перенесите папку проекта ```~\OSPanel\domains\```
- Заполните базу данных из файлы ```127_0_0_1.sql```
- Запустите проект
